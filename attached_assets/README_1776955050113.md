# HealthPro — Персональний цифровий щоденник здоров'я

> **Застосунок є допоміжним інструментом і не замінює професійну медичну консультацію.**
> Для точних замірів використовуйте сертифіковані тонометри.

---

## Про проєкт

**HealthPro** — застосунок для зручного моніторингу ключових показників організму та контролю за прийомом ліків. Допомагає систематизувати дані про самопочуття, вчасно помітити зміни та надати лікарю точну статистику.

Усі медичні дані зберігаються **виключно локально на вашому пристрої**. Жодних серверів, жодного хмарного зберігання, жодного стеження.

---

## Можливості

| Модуль | Опис |
|---|---|
| 📊 **Тиск і пульс** | Щоденні заміри з класифікацією за стандартами ВООЗ/МОЗ та графіками тенденцій |
| 💊 **Менеджер ліків** | Розклад прийому препаратів з дозуванням і нагадуваннями |
| 🧬 **Індекс здоров'я** | Персональна оцінка стану на основі тиску, пульсу, ІМТ та активності |
| 🚑 **Екстрені сповіщення** | SMS довіреній особі при критично високих показниках тиску |
| 📋 **Звіти для лікаря** | Експорт у PDF, CSV або JSON — для друку або надсилання в месенджері |
| 🚶 **Крокомір** | Відстеження щоденної активності через вбудований лічильник кроків |

## Запуск локально

### Вимоги

- [Node.js](https://nodejs.org/) v18+
- [pnpm](https://pnpm.io/) v8+
- [Python](https://python.org/) 3.11+

### Встановлення

```bash
# 1. Клонувати репозиторій
git clone https://github.com/chironkh-dev/HealthPro-Moie-Zdorovia.git
cd HealthPro-Moie-Zdorovia

# 2. Перейти до директорії застосунку
cd HealthPro-Moie-Zdorovia

# 3. Встановити залежності
pnpm install

# 4. Запустити в режимі розробки
pnpm dev --host 0.0.0.0 --port 5000
```

Застосунок буде доступний за адресою: `http://localhost:5000`

### Збірка для продакшну

```bash
pnpm build
# Результат у папці dist/
```

## Дані та приватність

HealthPro **не збирає і не передає** жодних даних:

- Усі записи (тиск, ліки, кроки, профіль) зберігаються у `localStorage` вашого браузера або локально на пристрої
- Застосунок не має серверної бази даних для даних користувачів
- Функція SMS для екстреного контакту використовує **стандартний SMS-клієнт вашого пристрою** — повідомлення не проходить через сторонні сервери
